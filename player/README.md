# 마비노기 MML 재생기 샘플

공개용 단일 페이지 샘플입니다.
외부 CDN이나 온라인 모듈을 사용하지 않고, 필요한 코드는 `js/` 폴더의 로컬 파일로 분리되어 있습니다.

## 구성

```text
mabinogi_mml_public/
├─ index.html
├─ styles.css
├─ README.md
├─ assets/
│  ├─ Roland_SC-55.sf2
│  └─ default-sf2-base64.js
└─ js/
   ├─ utils.js
   ├─ midi-to-mml.js
   ├─ mml-parser.js
   ├─ sf2-sampler.js
   └─ app.js
```

## 기능

- MML 재생 / 정지
- 반복 재생 체크박스, 기본값 꺼짐
- 시작 지점으로 되돌리기 버튼
- 재생 위치 확인 및 슬라이더 이동
- 재생 중 슬라이더 클릭/이동으로 구간 이동
- 재생 슬라이더에 템포 변화 지점 표시
  - 현재 적용 중인 템포는 초록색으로 크게 표시
- 볼륨 조절, 기본값 100%
- `mid`, `midi`, `txt`, `md` 파일 불러오기
  - `mid`, `midi`: MML로 변환 후 전체 MML에 반영
  - `txt`, `md`: `MML@...;` 형식일 때 전체 MML에 반영
- 기본 SF2 내장 사용: `Roland_SC-55.sf2`
- 사용자 SF2 선택 및 기본 사운드 복귀
- 전체 MML / 멜로디 / 화음1~5 탭 편집
- 현재 탭의 글자 수 표시
  - 전체 MML은 파트 내용 글자 수 합계만 표시합니다. `MML@`, 쉼표, 세미콜론은 제외합니다.
  - 글자 수는 `1,234 자` 형식으로 표시하며, 앞에 파트 이름은 붙이지 않습니다.
- 전체 MML에서 파트별 색상 표시
- 현재 탭의 MML 클립보드 복사
- 클립보드에서 MML 붙여넣기
- 현재 탭의 MML을 `.md` 파일로 저장
  - 지원 브라우저에서는 저장 위치와 파일명을 고르는 저장 다이얼로그를 사용합니다.
  - 미지원 브라우저에서는 파일명 입력 후 다운로드 방식으로 저장합니다.
- 전체 MML에서 규칙에 맞지 않는 문자/숫자 빨간 배경 표시
- 악보 공유 / 디스코드 바로가기 버튼

## 실행

폴더 전체를 서버에 올려 사용하세요.
로컬 확인은 아래처럼 실행할 수 있습니다.

```bash
cd mabinogi_mml_public
python -m http.server 8000
```

브라우저에서 `http://localhost:8000/`로 접속합니다.
`index.html`만 따로 복사하면 기본 SF2와 JS 파일을 찾지 못하므로 폴더 구조를 유지해야 합니다.

- 전체 MML 표시 영역은 빈 채널 콤마를 유지해 파트 순서를 보존합니다.
- 복사하기/저장하기는 빈 채널을 제거한 정돈된 MML을 사용합니다.

## 성능 최적화 메모

- 긴 MML을 재생할 때 모든 음표를 한 번에 예약하지 않고, 현재 재생 위치 기준 약 1.6초 앞까지만 예약하는 look-ahead 스케줄러를 사용합니다.
- SF2 샘플 버퍼와 음표별 재생 region은 재생 시작 전에 준비해 첫 재생 중 끊김을 줄입니다.
- V0처럼 소리 나지 않는 음표는 실제 오디오 노드를 만들지 않습니다.
- 64분음표처럼 짧은 음표는 attack/release 시간을 음 길이에 맞게 줄여 짧은 음이 뭉개지거나 밀리는 현상을 줄였습니다.
- 재생이 끝난 AudioBufferSourceNode는 자동으로 active 목록에서 제거해 장시간 재생 시 메모리 증가를 줄였습니다.


- 재생 시간 표시는 `분:초.밀리초` 형식으로 표시됩니다. 예: `01:23.456`
